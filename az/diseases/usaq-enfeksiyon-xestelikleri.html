Uşaq Enfeksion Xəstəlikləri

Uşaq enfeksion xəstəlikləri haqqında ətraflı məlumat və peşəkar müalicə yanaşmaları. Hansi şikayətlərlə müraciət edilməli və nəyə diqqət yetirmək lazımdır?

🔍 Əsas Şikayətlər 📞 Konsultasiya

Hansi Şikayətlərlə Müraciət Edilməlidir?

Uşaq enfeksion xəstəliklərinə dəlalət edən əsas simptom və şikayətlər haqqında məlumat

🌡️

Kəskin Şikayətlər

  • • Yüksək hərarət (38°C-dən yuxarı)
  • • Şiddətli baş ağrısı və boyun sərtliyi
  • • Tənəffüs çətinliyi və nəfəs darlığı
  • • Şiddətli qarın ağrısı və qusma
  • • Dəridə səpgilər və ləkələr

⚠️ Təcili müraciət tələb olunur

🤒

Ümumi Şikayətlər

  • • Orta dərəcəli hərarət (37.5-38°C)
  • • Zəiflik və yorğunluq
  • • İştahsızlıq və həzmsizlik
  • • Öskürək və burun axması
  • • Boğaz ağrısı və udma çətinliyi

⚡ Vaxtında müraciət tövsiyə olunur

🕐

Uzunmüddətli Şikayətlər

  • • Davamlı hərarət (2 həftədən çox)
  • • Çəki azalması və böyümə problemi
  • • Gecə tərləməsi və narahatlıq
  • • Tez-tez xəstələnmə
  • • Limfa düyünlərinin böyüməsi

📋 Ətraflı müayinə lazımdır

Əsas Xəstəlik Kateqoriyaları

Uşaq enfeksion xəstəliklərinin əsas növləri və xüsusiyyətləri

🫁

Tənəffüs Yolu İnfeksiyaları

Bronxit, pnevmoniya, angina və digər tənəffüs yolu xəstəlikləri

  • • Viral və bakterial angina
  • • Bronxit və bronxiolitis
  • • Pnevmoniya
  • • Sinusit və rinit
🍽️

Mədə-Bağırsaq İnfeksiyaları

Gastroenterit, qida zəhərlənməsi və digər həzm sistemi problemləri

  • • Viral gastroenterit
  • • Bakterial qida zəhərlənməsi
  • • Rotavirus infeksiyası
  • • Parazitar xəstəliklər
🦠

Dəri İnfeksiyaları

Bakterial və viral dəri xəstəlikləri, səpgilər və döküntülər

  • • El-ayaq-ağız xəstəliyi
  • • İmpetiqo
  • • Sellulit
  • • Viral səpgilər
🧠

Mərkəzi Sinir Sistemi İnfeksiyaları

Meningit, ensefalit və digər ciddi neyroinfeksiyalar

  • • Viral meningit
  • • Bakterial meningit
  • • Ensefalit
  • • Beyin absesi
🫘

Sidik Yolu İnfeksiyaları

Sistit, piyelonefrit və digər urogenital infeksiyalar

  • • Kəskin sistit
  • • Piyelonefrit
  • • Uretrit
  • • Təkrarlanan SYİ
👶

Anadangəlmə İnfeksiyalar

TORCH infeksiyaları və digər anadangəlmə xəstəliklər

  • • CMV infeksiyası
  • • Toksoplazmoz
  • • Rubella
  • • HSV infeksiyası

Nə Vaxt Həkimə Müraciət Etmək Lazımdır?

Uşaq enfeksion xəstəliklərində vaxtında müraciət çox vacibdir

🚨

Təcili Müraciət

  • • 40°C-dən yuxarı hərarət
  • • Tənəffüs çətinliyi
  • • Şüur pozğunluğu
  • • Qanlı qusma/ishal
  • • Boyun sərtliyi
⚡

Tez Müraciət

  • • 3 gündən çox hərarət
  • • Davamlı qusma
  • • Şiddətli baş ağrısı
  • • Dəridə səpgilər
  • • Susuzluq əlamətləri
📅

Planlı Müraciət

  • • Tez-tez xəstələnmə
  • • Böyümə problemləri
  • • Peyvənd konsultasiyası
  • • Profilaktik müayinə
  • • İmmunitet qiymətləndirilməsi

💡 Vacib Qeyd

Valideynlərin intuisiyası çox vacibdir. Uşağınızın normal davranışından fərqli bir şey hiss etsəniz, həkimə müraciət etməkdən çəkinməyin.
